# Team-Project-Archive

이 저장소는 진행했던 다양한 팀 프로젝트들을 정리하고 보관하기 위한 아카이브입니다.

## 📂 프로젝트 목록 (Project List)

이곳에 아카이빙된 프로젝트들의 목록입니다.

- **[5so4so](./5so4so)**: JSP와 Spring을 이용한 쇼핑몰 만들기 (다이소/오늘의집 컨셉)
- **[Chatting-Program](./Chatting-Program)**: JAVA 언어와 TCP 소켓통신을 이용한 채팅프로그램
- **[Movie Reservation Application](./movie_reservation_application)**: 크롤링을 통한 실시간 영화 정보 확인 및 예매 사이트 (BITBOX)
- **[Restaurant Rating System](./Restaurant-Rating-System)**: 식당의 이름과 대표메뉴, 평점을 저장하고 검색하는 콘솔 프로그램

## 🛠️ 기술 스택 (Tech Stack)

프로젝트 전반에 걸쳐 사용된 주요 기술들을 나열합니다.

- **Languages**: Java (18), JavaScript (ES6+), HTML5, CSS3
- **Frameworks & Libraries**: Spring Framework (5.3.20), MyBatis (3.5.3), JSP (2.1), Servlet, JQuery (3.6.0)
- **Tools & Infra**: Git, MariaDB, Heroku, NAS
- **Key Concepts**: TCP Socket, Crawling, AJAX, File I/O
